# blogdoktersobri (React + Vite + Tailwind + Vercel)

Starter blog kesehatan SEO-friendly tanpa Next.js.

## Jalankan lokal
```bash
npm install
npm run dev
```

## Deploy ke Vercel
- Import repo ke Vercel
- Framework preset: Vite
- Build: `npm run build`
- Output: `dist`

## Catatan
- Konten bersifat edukatif, bukan diagnosis medis.
- Lengkapi `ads.txt` dengan publisher ID AdSense Anda.
